# socket-in-C

A simple, client-server chat application built with C sockets. This application demonstrates basic network programming concepts with visual separation.


## Requirements

- GCC compiler
- Linux/Unix system (tested on Linux)
- Standard C libraries (stdio, stdlib, string, time, sys/socket, netinet/in, arpa/inet)

## Compilation

Compile the server and client applications:

```bash
# Compile server
gcc -o server server.c

# Compile client
gcc -o client client.c
```
